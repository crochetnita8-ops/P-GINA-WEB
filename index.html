¡Bienvenid@!

Bienbenid@ a Crochet Nita, un humilde emprendimiento con finalidad de dar a conocer el arte del crochet.

Dejanos contarte como empezó nuestra historia:

Para ello nos remontamos a 2008, María (creadora de Crochet Nita) necesitaba un arrullo para su hija que estaba en camino, Nadia (creadora de esta página web con sus 17 años ya!!!), pero las prendas para bebés eran muy costosas. Así pues María se fue a una antigua tienda de Balaguer y compró una revista, lana y una aguja de crochet, aprendió a hacer punto granny y consiguio hacerle un arrullo para el nacimiento de Nadia.

Con el paso del tiempo con mayor o menor intensidad María nunca dejó de tejer. En 2023 comenzó a innovar haciendo prendas de vestir, cada vez con más soltura. Después de númerosas peticiones, María decidió crear una página web para sus creaciones.
